安装Twurl并调试extended tweet mode

时间： 2018-02-08   |   分类： DataBuilder     |   阅读： 807 字 ~4分钟

在研究tweet字节限制由140字节变为280字节时，想要玩一玩Twitter API控制台工具(https://dev.twitter.com/rest/tools/console)，结果dev.twitter.com跳到了developer.twitter.com，这才发现Twitter REST API按照Standard、Premium和Enterprise划分开始走上收费的道路。 Twitter API控制台工具已经找不到了，官方示例使用twurl命令工具完成的。

安装Twurl

  1. 在https://rubyinstaller.org/下载并安装RubyInstaller
  2. 执行gem install twurl

调试extended tweet mode

准备环境

认证

twurl authorize --consumer-key key --consumer-secret secret

搜索tweet

twurl "/1.1/search/tweets.json?q=scnydq"
