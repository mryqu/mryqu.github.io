搭建Activiti的IntelliJ IDEA开发调试环境

时间： 2015-02-09   |   分类： workflow     |   阅读： 90 字 ~1分钟

准备工作

在搭建Activiti的IntelliJ IDEA开发调试环境前，确保下列软件已经安装： - JDK 1.6 - ant - Maven - IntelliJ IDEA

导入Activiti

Import Project

通过Activiti的Maven pom.xml导入IntelliJ IDEA项目。

Import Modules

导入IntelliJ IDEA项目后，仅有几个Activiti模块变成了IntelliJIDEA项目中的模块。为了可以调试所有模块，通过ProjectStructure菜单简单粗暴地将剩余的Activiti模块导入IntelliJ模块。 对每个手工导入的模块设置源代码、资源、测试代码和测试资源。

Frameworks Dection

手工导入所有模块后，重启IntelliJ IDEA，它会自动检测所有模块的类型，例如Spring、WEB、JPA和GWT等。

构建Activiti

Activiti项目可以通过下面的ant命令构建：

.....\wfgitws\Activiti\distro>ant -Dnodocs=true clean distro
