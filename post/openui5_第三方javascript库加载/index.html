[OpenUI5] 第三方JavaScript库加载

时间： 2015-02-06   |   分类： FrontEnd     |   阅读： 27 字 ~1分钟

SAP often put 3rd JavaScript libraries at \resources\sap\ui\thirdparty, then load as below:

jQuery.sap.require("sap/ui/thirdparty/d3");
